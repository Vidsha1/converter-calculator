Currency Conversion
  • This Module helps to convert currency from one to another.
  • E.g. Consider if it is 100 ZAR to be converted to USD.
  • It also states the historic trends.
Length Conversion
  • This Module helps to convert length unit from one to another.
  • E.g. Consider if it is 100 Metre to be converted to Centi-Meter.
  • It also gives you feature of adding the new unit.
